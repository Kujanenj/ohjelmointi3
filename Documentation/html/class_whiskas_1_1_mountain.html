Detailed Description

The Mountain class represents Mountain in the gameworld.

Forest has BasicProduction:

  • Money = 1
  • Food = 3
  • Wood = 5
  • Stone = 1
  • Ore = 0

Building in the forest takes time. So buildings get extra hold-marker.

Tile supports 2 buildings.
